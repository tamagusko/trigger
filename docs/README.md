## What is This Platform?

The TRIGGER platform lets you create and deliver online courses, like Coursera or edX.

**You can:**
- Create video lessons
- Add quizzes and assignments
- Enroll students
- Track their progress
- Issue certificates

**Built with:**
- Open edX (same technology as MIT and Harvard's edX.org)
- Docker (runs on any computer)
- Tutor (makes setup easy)

## Quick Commands

**Start platform:**
```bash
tutor local start -d
```

**Stop platform:**
```bash
tutor local stop
```

**Check status:**
```bash
tutor local status
```

**Access platform:**
- Studio (create courses): http://studio.local.overhang.io
- LMS (student view): http://local.overhang.io
